Релиз 2.0

Выпуск релиза 2.0 -- 20 августа 2008г. (Основная тема: описана процессная сетка ISO 15288, и в ней указаны различные оргтехнологии).

Истории Релиза:

  • для каждого процесса системной инженерии (ISO 15288) на первой странице есть кратенькое его описание вместе с рекомендуемыми технологиями.
  • есть описание DEMO (впрочем, уже какое-то есть)
  • есть описание технологий интеграции данных
  • факультативно: для отдельных тем есть слайдовые презентации

Релиз 1.01

Выпуск релиза 1.01 -- 20 августа 2007г. (Основная тема: определить scope)

Истории релиза:

  • есть очень грубые черновики Рассказов по основным мощным идеям
  • есть очень грубые черновики Рассказов по различным темам PraxOS
  • работает двухоконный редактор планов (Squeak-Morphic), его код в monticello
  • отрефлектированы и уточнены цели PraxOS
  • скорректирована структура сайта PraxOS

Релиз 1.0

Выпуск релиза 1.0 -- 28 июля 2007г. (Основная тема: действия, цели, планы, графики/расписания).

Истории релиза:

  • <выполнено>про любые планы и их выполнение можно рассказать на русском языке -- в объеме терминологии TAEMS, и об этом есть тексты на сайте
  • <уточнено и перенесено в 1.01>любые планы и их выполнение можно описать в софте -- в объеме модели TAEMS, код софта раскрыт (в репозитории monticello)
  • <перенесено в следующие релизы>есть план учебного семинара, который позволяет освоить понятийный аппарат модели TAEMS (опубликован на сайте)
  • <выполнено>зайдя на сайт http://praxos.ru люди смогут разобраться, чем это тут занимаются и как организационную систему PraxOS можно будет употребить.
